== Description ==
Clear Booking / Reservation system by Unified Digital Media is designed for travel accommodation provider that want to offer their guest more convenience when making a booking. The booking / reservation system allows travellers to choose their rooms as well as available promotional offers and amenities that would make their stay more comfortable. Whilst allow easy checkout using the PayPal API.

 
Clear Booking creates an easy to manage booking for guests as all their choices are bundled into one convenient location and totalled, giving them full transparency of their booking. Using this WordPress Booking / Reservation plugin, lets travel accommodation providers you can add a places of interest filter and map on any page.

Ordered list:

1. PayPal payment gateway
1. Stand alone reservation , booking details, booking confirmation , thank you and booking cancellation pages
1. Itemised billing

Unordered list:

* Reservation form on any page using wordpress shortcode
* Automatic reservation dates validation
* Tax calculations and sub-totals including discounts and amenities

== Installation ==
1. Upload \"test-plugin.php\" to the \"/wp-content/plugins/\" directory.
1. Activate the plugin through the \"Plugins\" menu in WordPress.
1. Place \"do_action( \'plugin_name_hook\' );\" in your templates.
